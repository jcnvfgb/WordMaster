# Проект WordMaster

Этот проект нацелен на увеличение скорости запоминания слов и помогает выучить новые.
Вы можете создавать свои собственные наборы слов.

Для запуска этого проекта необходимо скачать Android Studio и настроить эмуляцию Android. Желательно использовать Android 10 Q Version или выше, так как все тесты проводились на этой версии.
После этого откройте окно "info" и введите команду "Leng", затем нажмите кнопку "EXECUTE".
Теперь вы можете пользоваться приложением.
